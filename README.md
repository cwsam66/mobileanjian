# 辅助器
- 只适用于安卓模拟器，千万别安装在手机
- 建议使用逍遥模拟器（Memu Play）

# 模拟器设置
- 屏幕分辨率
  - 宽:1280 
  - 高：720 
  - DPI:240

# 简单教程
- 安装辅助器
- 打开辅助器，后直接启动辅助
- 进入游戏
- 点击辅助器浮动图标
- 打开设置，一下功能只能选一个执行
  - 日常看板任务
  - 日程商会任务 （新更新的宠物任务不能进行）
  - 钓鱼 （需要先如钓鱼界面）
  - 园艺 (自动验证码部分还未解决)
- 做好选项后保存设置再点击启动便可
